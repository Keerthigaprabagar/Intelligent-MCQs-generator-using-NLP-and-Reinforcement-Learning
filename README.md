# Intelligent-MCQs-generator-using-NLP-and-Reinforcement-Learning
This system allows users to upload any document, such as textbooks, document, or PDF and generates high-quality MCQs tailored to the content.  It incorporates a feedback loop for continuous improvement and customization, enabling educators to adapt questions based on learning goals and difficulty levels.
Intelligent MCQ generation system leverages:
Natural Language Processing (NLP): To preprocess and extract meaningful insights from documents.
Reinforcement Learning (PPO): To dynamically improve question quality through user feedback.
Large Language Models (LLM) - GPT-4: To generate contextually relevant, challenging, accurate questions.
